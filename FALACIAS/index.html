Vídeoinstalación, 2011
5’ // Color // Sin sonido // Con subtítulos
La condición narrativa, La Capella, Barcelona, 2012; Festival Art Souterrain, Montreal, 2013


Primera parte del díptico elaborado a partir del texto Especies de Espacios, de Georges Perec. A partir de fragmentos de este texto, la autora reinterpreta un espacio abstracto y sus componentes mediante un único plano, el del suelo de una vieja terraza, un travelling cámara en mano que podríamos comparar con un mapa cartográfico.
