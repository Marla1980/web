(De la serie: Elucubraciones en torno a esa serie de características que debería tener –o no– la literatura del futuro)

Vídeo monocanal // 15’32’’ // 2014
Retrat de l’ artista adolescent, Bòlit, Centre d’ Art Contemporani de Girona, 2014 // Beca Bòlit Mentor
Fairy Tales (Can Palauet, Mataró, 2014 / Tinglado 2, Tarragona, 2015 / Espai Moritz, Cornellà, 2015 / Sala Muncunill, Terrassa, 2016 / Centre Cultural El Casino, Manresa, 2017)
L’Alternativa, Festival de Cinema Independent de Barcelona, 2014 / MOV Festival, Festival de Cine Universitario de Pernambuco, 2014 / XVI Concurso Internacional Encuentros de Arte Contemporáneo, Alicante, 2016 // Festival Panoramic, Granollers, 2017

Tragicomedia en cinco actos en la que realizo una imposible comparación entre la infancia de Stephen Dedalus (alter ego del escritor James Joyce) y la mía propia. Ejercicio metalingüístico que reflexiona sobre las diferentes maneras de contar una historia y hacerla verosímil aun a pesar de su imposibilidad.
